# Sublime Text

Personal configuration for the text editor Sublime Text ([https://www.sublimetext.com/](sublimetext.com/))

## Themes / Color Schemes

* Theme: Default
* Color scheme is set to `auto`:
  * Light scheme: `Breakers`
  * Dark scheme: `Dracula`

## Plugins

* BracketHighlighter
* Copy Relative Path
* Dracula Color Scheme
* Emmet
* Package Control
